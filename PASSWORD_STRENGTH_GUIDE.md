# 密码强度提示 - 实时反馈效果说明

## 功能特性

### 1. 始终显示提示框

用户点击密码输入框时，**立即显示**密码要求提示，无需等待输入。

### 2. 实时反馈

用户每输入一个字符，提示框**立即更新**，显示哪些要求已满足，哪些未满足。

## 技术实现

### 实时检查触发

```javascript
// 输入时触发
@input="checkPasswordStrength"

// 焦点时触发
@focus="passwordFocused = true"
```

### 强度计算

```javascript
checkPasswordStrength() {
  // 检查各项要求
  hasMinLength = password.length >= minPasswordLength
  hasLowerCase = /[a-z]/.test(password)
  hasUpperCase = /[A-Z]/.test(password)
  hasNumber = /\d/.test(password)
  hasSpecialChar = /[!@#$%^&*()_+\-=\[\]{}|;:,.<>?]/.test(password)

  // 计算满足的条件数量
  metCount = 满足的要求数量

  // 设置强度等级和进度
  if (metCount <= 2) → 弱 (20%)
  if (metCount === 3) → 中等 (50%)
  if (metCount === 4) → 良好 (75%)
  if (metCount === 5) → 强 (100%)
}
```

### 条件渲染

```vue
<!-- 未输入时显示提示 -->
<div v-if="!registerForm.password" class="password-hint">
  <i class="el-icon-info"></i>
  <span>请输入密码，密码需满足以下要求：</span>
</div>

<!-- 输入后显示强度条 -->
<div v-else>
  <div class="strength-bar">...</div>
  <div class="strength-text">...</div>
</div>

<!-- 要求列表始终显示 -->
<div class="strength-requirements">...</div>
```
