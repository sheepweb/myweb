## 📋 文档说明

### Clash 配置系统概述

Clash 配置系统使用模板驱动的方式生成订阅配置：

1. **模板文件**: `uploads/config/temp.yaml`
   - 包含 16 个代理组
   - 包含 3376 条分流规则
   - 包含优化的 DNS 配置

2. **生成流程**:
   - 读取模板文件
   - 注入实际节点列表
   - 更新代理组配置
   - 保留所有自定义规则
   - 返回完整配置

3. **特性**:
   - ✅ 支持多种代理协议（VMess, VLess, Trojan, SS, SSR, Hysteria 等）
   - ✅ 支持多种代理组类型（select, url-test, fallback, load-balance）
   - ✅ 自动 Emoji 字符处理
   - ✅ 完整的规则保留
