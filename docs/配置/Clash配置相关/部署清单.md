## 🆘 故障排除

### 问题 1: 配置不生效
**原因**: 可能是旧配置被缓存  
**解决**: 删除 uploads/config/ 下的缓存文件，重启服务

### 问题 2: 编译失败
**原因**: Go 版本或依赖问题  
**解决**: 
```bash
go mod tidy
go build -o bin/server ./cmd/server/main.go
```

### 问题 3: 订阅返回旧配置
**原因**: 模板文件路径错误或权限问题  
**解决**: 
```bash
ls -lh uploads/config/temp.yaml
chmod 644 uploads/config/temp.yaml
```

### 问题 4: 代理组不完整
**原因**: 模板格式错误  
**解决**: 运行验证工具
```bash
python3 scripts/verify_clash_config.py
```
