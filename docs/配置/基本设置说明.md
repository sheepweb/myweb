# 基本设置说明

## 概述

基本设置用于配置网站的基本信息、外观、客服联系方式等。这些设置会影响用户界面显示和系统行为。

## 配置位置

- **管理员后台** → **系统设置** → **基本设置**

## 一、网站基本信息

### 1. 网站名称

- **说明**：显示在网站标题、浏览器标签页、邮件通知等位置
- **示例**：`CBoard`、`我的代理服务`、`VPN 管理平台`
- **建议**：简洁明了，易于识别

### 2. 网站描述

- **说明**：网站简介或 slogan，显示在首页、帮助页面等位置
- **示例**：`现代化的代理服务管理平台`、`安全、快速、稳定`
- **建议**：简短描述网站功能或特点

### 3. 网站域名

- **说明**：用于生成订阅地址和邮件链接中的域名
- **格式**：`example.com`（不需要 `http://` 或 `https://`）
- **留空**：如果留空，系统会使用请求的域名

**使用场景：**
- 生成订阅地址：`https://your-domain.com/subscribe/xxx`
- 邮件中的链接：`https://your-domain.com/reset-password?token=xxx`

**注意事项：**
- 确保域名已正确解析到服务器
- 建议填写，避免使用临时域名或 IP 地址

### 4. 网站 Logo

- **说明**：网站 Logo 图片，显示在页面顶部、登录页等位置
- **格式**：支持常见图片格式（JPG、PNG、GIF 等）
- **大小限制**：建议不超过 2MB
- **建议尺寸**：200x50 像素或按比例缩放

**上传方法：**
1. 点击 Logo 上传区域
2. 选择图片文件
3. 系统自动上传并显示

## 二、主题设置

### 1. 默认主题

- **选项**：
  - **浅色主题**：适合日间使用
  - **深色主题**：适合夜间使用，护眼
  - **跟随系统**：根据用户系统设置自动切换
- **说明**：新用户首次访问时使用的主题

**建议：**
- 公开服务：使用「跟随系统」或「浅色主题」
- 专业服务：根据品牌风格选择

## 三、客服联系方式

### 1. 售后 QQ

- **说明**：显示在帮助中心，用户可以通过 QQ 联系客服
- **格式**：QQ 号码（纯数字）
- **留空**：如果留空，帮助中心不显示 QQ

**使用场景：**
- 帮助中心显示 QQ 联系方式
- 用户可以通过 QQ 咨询问题

### 2. 售后邮箱

- **说明**：显示在帮助中心，用户可以通过邮件联系客服
- **格式**：`support@example.com`
- **留空**：如果留空，帮助中心不显示邮箱

**使用场景：**
- 帮助中心显示邮箱联系方式
- 用户可以通过邮件咨询问题

## 四、用户界面设置

### 1. 统一认证页面

- **开启**：使用集成登录页（登录和注册在同一页面）
- **关闭**：使用传统分离页面（登录和注册分开）

**区别：**
- **统一页面**：用户体验更流畅，适合现代 Web 应用
- **分离页面**：传统方式，适合需要明确区分登录和注册的场景

## 五、GeoIP 数据库管理

### 1. 数据库状态

系统会显示 GeoIP 数据库的状态：
- **已启用/未启用**：数据库是否可用
- **文件大小**：数据库文件大小（约 60-80MB）
- **更新时间**：数据库最后更新时间

### 2. 更新 GeoIP 数据库

- **功能**：下载最新的 GeoIP 数据库，用于 IP 地址地理位置解析
- **用途**：登录历史、数据分析中的地区信息显示
- **文件大小**：约 60-80MB，下载可能需要几分钟

**更新方法：**
1. 点击「更新 GeoIP 数据库」按钮
2. 系统自动下载最新数据库
3. 更新完成后显示新的更新时间

**注意事项：**
- 更新需要网络连接
- 文件较大，下载可能需要时间
- 建议定期更新（每月一次）

## 六、配置示例

### 示例 1：标准配置

```
网站名称：CBoard
网站描述：现代化的代理服务管理平台
网站域名：example.com
默认主题：跟随系统
售后 QQ：123456789
售后邮箱：support@example.com
统一认证页面：✅ 开启
```

### 示例 2：简洁配置

```
网站名称：我的服务
网站描述：安全、快速、稳定
网站域名：（留空，使用请求域名）
默认主题：浅色主题
售后 QQ：（留空）
售后邮箱：support@example.com
统一认证页面：❌ 关闭
```

## 七、注意事项

1. **网站域名**：建议填写，确保订阅地址和邮件链接正确
2. **Logo 上传**：确保图片大小和格式符合要求
3. **GeoIP 更新**：定期更新以获取最新地理位置信息
4. **客服联系方式**：至少填写一种联系方式，方便用户咨询

## 八、常见问题

### Q1: Logo 上传失败？

**检查项：**
- 图片大小是否超过 2MB
- 图片格式是否支持（JPG、PNG、GIF 等）
- 服务器磁盘空间是否充足

### Q2: 订阅地址中的域名不正确？

**解决方法：**
- 在「网站域名」中填写正确的域名
- 确保域名已正确解析到服务器
- 清除浏览器缓存后重试

### Q3: GeoIP 数据库更新失败？

**检查项：**
- 网络连接是否正常
- 服务器磁盘空间是否充足（需要至少 100MB）
- 查看系统日志中的错误信息
