# 客户通知设置说明

## 概述

客户通知设置用于控制发送给用户的邮件通知，包括系统通知、订阅到期提醒、新订单通知等。合理配置通知设置可以提高用户体验和用户留存率。

## 配置位置

- **管理员后台** → **系统设置** → **通知设置** → **客户通知**

## 一、通知开关

### 1. 系统通知

- **开启**：启用所有系统通知
- **关闭**：禁用所有系统通知（不影响其他通知设置）

**说明：**
- 这是总开关，关闭后所有客户通知都不会发送
- 建议保持开启，确保用户能及时收到重要通知

### 2. 邮件通知

- **开启**：启用邮件通知功能
- **关闭**：禁用邮件通知（其他通知方式不受影响）

**前提条件：**
- 必须正确配置邮件服务器（见 [邮件服务器配置说明](./邮件服务器配置说明.md)）
- 确保 SMTP 服务正常，否则邮件无法发送

## 二、通知类型

### 1. 订阅到期提醒

- **开启**：订阅即将到期时，向用户发送提醒邮件
- **关闭**：不发送到期提醒

**触发时机：**
- 订阅到期前 7 天
- 订阅到期前 3 天
- 订阅到期前 1 天
- 订阅已到期

**邮件内容：**
- 订阅到期时间
- 续费链接
- 订阅详情

**建议：**
- 建议开启，提醒用户及时续费
- 可以提高用户留存率和续费率

### 2. 新用户注册通知

- **开启**：新用户注册成功后，发送欢迎邮件
- **关闭**：不发送注册通知

**邮件内容：**
- 欢迎信息
- 账号信息
- 使用指南链接

**建议：**
- 建议开启，提升用户体验
- 可以帮助新用户快速了解如何使用服务

### 3. 新订单通知

- **开启**：用户下单后，发送订单确认邮件
- **关闭**：不发送订单通知

**邮件内容：**
- 订单信息（订单号、金额、套餐等）
- 支付链接（如果未支付）
- 订单状态

**建议：**
- 建议开启，让用户及时了解订单状态
- 方便用户完成支付

## 三、通知配置示例

### 示例 1：完整通知配置（推荐）

```
系统通知：✅ 开启
邮件通知：✅ 开启
订阅到期提醒：✅ 开启
新用户注册通知：✅ 开启
新订单通知：✅ 开启
```

**适用场景：** 大多数场景，确保用户能及时收到所有重要通知。

### 示例 2：仅重要通知

```
系统通知：✅ 开启
邮件通知：✅ 开启
订阅到期提醒：✅ 开启
新用户注册通知：❌ 关闭
新订单通知：✅ 开启
```

**适用场景：** 减少邮件发送频率，只发送重要通知。

### 示例 3：禁用所有通知

```
系统通知：❌ 关闭
邮件通知：❌ 关闭
订阅到期提醒：❌ 关闭
新用户注册通知：❌ 关闭
新订单通知：❌ 关闭
```

**适用场景：** 测试环境或不需要邮件通知的场景。

## 四、通知邮件说明

### 1. 邮件模板

系统使用预设的邮件模板发送通知，包括：

- **注册欢迎邮件**：欢迎新用户，介绍服务
- **订阅到期提醒**：提醒用户订阅即将到期
- **订单确认邮件**：确认订单信息
- **订阅重置邮件**：通知用户订阅已重置
- **密码重置邮件**：密码重置链接

### 2. 邮件内容

邮件内容包含：
- **标题**：通知类型和简要信息
- **正文**：详细信息和操作链接
- **签名**：系统签名和联系方式

### 3. 邮件发送

- **发送时机**：事件触发时立即发送
- **发送方式**：通过配置的 SMTP 服务器发送
- **发送状态**：系统会记录发送状态（成功/失败）

## 五、注意事项

1. **邮件服务器配置**：必须正确配置邮件服务器，否则通知无法发送
2. **邮件发送频率**：避免过于频繁的通知，影响用户体验
3. **邮件内容**：确保邮件内容准确、友好、有用
4. **垃圾邮件**：注意邮件可能被标记为垃圾邮件，建议使用专业的 SMTP 服务

## 六、常见问题

### Q1: 用户收不到通知邮件？

**检查项：**
- 邮件服务器配置是否正确（见 [邮件服务器配置说明](./邮件服务器配置说明.md)）
- 通知开关是否已开启
- 用户邮箱是否正确
- 邮件是否被垃圾邮件拦截
- 查看系统日志中的发送记录

### Q2: 如何自定义邮件模板？

**说明：**
- 邮件模板在系统代码中定义
- 如需自定义，需要修改代码中的模板文件
- 建议联系开发人员或参考系统文档

### Q3: 可以发送短信通知吗？

**说明：**
- 当前系统主要支持邮件通知
- 短信通知需要集成短信服务商 API
- 如需短信通知，需要开发支持

### Q4: 如何测试通知功能？

**方法：**
- **注册通知**：注册一个新账号，检查是否收到欢迎邮件
- **到期提醒**：手动修改订阅到期时间，触发到期提醒
- **订单通知**：创建一个测试订单，检查是否收到确认邮件
