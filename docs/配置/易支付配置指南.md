# 易支付配置指南

## 概述

本指南包含易支付的完整配置说明和通用适配器系统介绍。

---

## 一、基础配置

### 1. 配置信息说明

根据易支付后台提供的信息，您需要配置以下内容：

#### 基础信息
- **网关地址**：您的易支付站点地址（例如 `https://your-pay-domain.com`）
- **商户编号 (pid)**：从易支付后台获取
- **MD5密钥**：从易支付后台获取

### 2. 签名方式

易支付支持三种签名方式：
- **MD5签名**：使用 MD5 密钥进行签名（最简单）
- **RSA签名**：使用 RSA 密钥对进行签名（更安全）
- **MD5+RSA签名**：同时使用 MD5 和 RSA（推荐，最安全）

### 3. 平台公钥和商户公钥说明

#### 平台公钥（Platform Public Key）
- **来源**：由易支付平台提供，在易支付后台 → 个人资料 → API信息 → RSA密钥 → 平台公钥中查看
- **用途**：验证易支付发送给您的回调通知的签名
- **填写位置**：系统后台支付配置中「易支付（统一配置）」→「平台公钥（RSA签名）」

#### 商户公钥（Merchant Public Key）
- **来源**：由您自己生成（OpenSSL 或在线工具）
- **用途**：签名您发送给易支付的请求
- **填写**：公钥内容填到易支付后台「商户公钥」；对应私钥填到系统后台「商户私钥（RSA签名）」字段

---

## 二、配置步骤

### 方式一：使用 MD5 签名（最简单）

1. 登录系统后台，进入「支付配置」
2. 选择「易支付（统一配置）」或新建支付方式
3. 填写：商户ID、商户密钥（MD5）、API 地址、签名方式选「MD5签名」、勾选需要的支付方式（支付宝、微信、QQ 等）

### 方式二：使用 MD5+RSA 签名（推荐）

1. 登录系统后台，进入「支付配置」
2. 选择「易支付（统一配置）」
3. 填写：商户ID、商户密钥、API 地址、签名方式选「MD5+RSA签名」、平台公钥（从易支付后台复制）、商户私钥（您生成的 RSA 私钥）、勾选支付方式

### 生成 RSA 密钥对（使用 RSA 时）

```bash
# 生成私钥
openssl genrsa -out merchant_private_key.pem 2048

# 生成公钥
openssl rsa -in merchant_private_key.pem -pubout -out merchant_public_key.pem
```

- 将**公钥内容**复制到易支付后台 → API信息 → RSA密钥 → 商户公钥
- 将**私钥内容**复制到系统后台支付配置的「商户私钥（RSA签名）」字段

### 回调地址配置

- **异步回调地址**：`https://您的域名/api/v1/payment/notify/yipay`
- **同步回调地址**：`https://您的域名/api/v1/payment/success`

## 三、如何使用易支付

1. **后台配置**：按上文在「支付配置」中完成易支付（或易支付-支付宝/微信/QQ 等）的网关、商户号、密钥、回调地址配置并保存。
2. **套餐与订单**：用户在站内选择套餐并下单时，可选择易支付或对应支付方式。
3. **支付流程**：用户跳转到易支付页面完成支付，支付成功后易支付会请求上述异步回调，系统自动将订单标记为已支付并更新用户订阅/余额。
4. **验证**：建议先创建小额测试订单，完成支付后检查订单状态是否自动更新，并查看系统日志确认回调签名验证通过。

---

## 四、通用适配器与多平台

系统可根据网关地址自动识别常见易支付平台（如 FH易支付、易支付官方、EZ易支付等）。若使用标准 API，只需正确填写网关地址和密钥即可，无需改代码。

在 `config_json` 中可配置：
- `gateway_url`：易支付网关地址（系统会自动拼接 `/mapi.php`）
- `api_url`：完整 API 地址（若已配置则优先使用）
- `platform_name`：可选，手动指定平台类型

## 五、常见问题

- **平台公钥如何填写？** 从易支付后台复制平台公钥，粘贴到系统配置即可。
- **商户公钥如何填写？** 自己生成 RSA 密钥对，公钥填到易支付后台，私钥填到系统后台。
- **仅用 MD5 签名**：只需填写商户 ID 和 MD5 密钥，无需填 RSA 相关字段。
- **签名验证失败**：检查签名方式、密钥是否完整无多余空格换行、回调地址是否为 HTTPS 且可访问，并查看系统日志中的具体错误。

---

## 六、安全建议

1. 妥善保管商户私钥和 MD5 密钥，不要泄露
2. 回调地址必须使用 HTTPS
3. 建议定期更换密钥
4. 若易支付支持，建议使用 MD5+RSA 签名
