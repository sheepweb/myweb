## 一、前端优化（立即可实施）

### 1. 路由懒加载 ⭐⭐⭐⭐⭐
**效果**: 首屏加载速度提升 50-70%

**当前问题**: 所有页面一次性加载
**优化方案**: 按需加载路由组件

```javascript
// ❌ 当前方式（全部导入）
import Users from '@/views/admin/Users.vue'
import Orders from '@/views/admin/Orders.vue'

// ✅ 优化方式（懒加载）
const Users = () => import('@/views/admin/Users.vue')
const Orders = () => import('@/views/admin/Orders.vue')
```

**实施步骤**:
```bash
# 修改 frontend/src/router/index.js
# 将所有 import 改为动态 import
```

**预期收益**:
- 首屏加载时间: 3s → 1s
- 首屏资源大小: 1.2MB → 300KB
- 用户体验: 显著提升

### 6. 请求优化 ⭐⭐⭐⭐⭐
**效果**: 接口响应速度提升 50-80%

#### 6.1 请求合并
```javascript
// ❌ 当前方式（多次请求）
const users = await api.get('/users')
const stats = await api.get('/stats')
const config = await api.get('/config')

// ✅ 优化方式（批量请求）
const [users, stats, config] = await Promise.all([
  api.get('/users'),
  api.get('/stats'),
  api.get('/config')
])
```

#### 6.2 请求缓存
```javascript
// 添加请求缓存
const cache = new Map()

api.interceptors.request.use(config => {
  if (config.cache) {
    const key = config.url + JSON.stringify(config.params)
    const cached = cache.get(key)
    if (cached && Date.now() - cached.time < 60000) {
      return Promise.resolve(cached.data)
    }
  }
  return config
})
```

#### 6.3 防抖节流
```javascript
// 搜索框防抖
const searchUsers = debounce(async (keyword) => {
  const result = await api.get('/users', { params: { keyword } })
}, 300)

// 滚动加载节流
const loadMore = throttle(async () => {
  const result = await api.get('/users', { params: { page: page++ } })
}, 1000)
```

```go
    subscription := getSubscription(user.ID) // 每次查询

// ✅ 预加载
db.Preload("Subscription").Find(&users)
```

#### 1.3 分页查询
```go
// 添加分页
func GetUsers(page, pageSize int) ([]User, int64, error) {
    var users []User
    var total int64

    offset := (page - 1) * pageSize

    db.Model(&User{}).Count(&total)
    db.Limit(pageSize).Offset(offset).Find(&users)

    return users, total, nil
}
```

### 2. Redis 缓存 ⭐⭐⭐⭐⭐
**效果**: 接口响应速度提升 90-95%

**已实现**: ✅ 你已经有 Redis 缓存

**优化建议**:

#### 2.1 增加缓存覆盖
```go
// 用户信息缓存
func GetUserByID(id uint) (*User, error) {
    cacheKey := fmt.Sprintf("user:%d", id)

    // 先查缓存
    if cached, err := redis.Get(cacheKey); err == nil {
        var user User
        json.Unmarshal([]byte(cached), &user)
        return &user, nil
    }

    // 查数据库
    var user User
    db.First(&user, id)

    // 写入缓存（5分钟）
    data, _ := json.Marshal(user)
    redis.Set(cacheKey, data, 5*time.Minute)

    return &user, nil
}
```

#### 2.2 缓存预热
```go
// 系统启动时预热热点数据
func WarmupCache() {
    // 预热套餐列表
    packages := GetAllPackages()
    cachePackages(packages)

    // 预热系统配置
    config := GetSystemConfig()
    cacheConfig(config)
}
```

### 4. 并发处理 ⭐⭐⭐⭐
**效果**: 接口响应速度提升 50-70%

```go
// ❌ 串行处理
user := getUser(id)
subscription := getSubscription(id)
orders := getOrders(id)

// ✅ 并发处理
var wg sync.WaitGroup
var user User
var subscription Subscription
var orders []Order

wg.Add(3)

go func() {
    defer wg.Done()
    user = getUser(id)
}()

go func() {
    defer wg.Done()
    subscription = getSubscription(id)
}()

go func() {
    defer wg.Done()
    orders = getOrders(id)
}()

wg.Wait()
```

### 1. CDN 加速 ⭐⭐⭐⭐⭐
**效果**: 静态资源加载速度提升 80-90%

**推荐方案**:

- **国外**: Cloudflare (免费)

**配置步骤**:
1. 将 `frontend/dist` 上传到 CDN
2. 修改 `index.html` 中的资源路径
3. 配置 CDN 缓存规则

```nginx
# Nginx 配置
location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
    expires 1y;
    add_header Cache-Control "public, immutable";
}
```

```nginx
# Nginx 缓存配置
proxy_cache_path /var/cache/nginx levels=1:2 keys_zone=api_cache:10m max_size=1g inactive=60m;
```
