## ⚠️ 警告问题

### 1. 已弃用的 `rand.Seed` 使用

**位置**：`internal/utils/order.go:11`

**问题**：
```go
rand.Seed(time.Now().UnixNano())  // 已弃用
```

**影响**：Go 1.20+ 已弃用 `rand.Seed`，应该使用 `rand.New(rand.NewSource(seed))`。

**修复状态**: ✅ 已修复
