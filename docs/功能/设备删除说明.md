# 设备删除说明

## 概述

设备删除功能用于移除用户订阅下的设备记录。删除设备后，该设备将无法继续使用订阅服务，设备数量会相应减少。系统支持用户自己删除设备和管理员删除设备两种方式。

## 一、用户自己删除设备

### 1. 访问设备管理页面

**路径：**
- **用户端** → **设备管理**

**说明：**
- 用户登录后，可以在用户中心找到「设备管理」菜单
- 页面显示用户所有订阅下的设备列表

### 2. 查看设备信息

设备列表显示以下信息：
- **设备名称**：自动识别的设备名称（如：Shadowrocket - iPhone 15 Pro - iOS 17.0）
- **设备类型**：移动设备/桌面设备/平板设备
- **软件信息**：使用的代理软件名称和版本
- **操作系统**：操作系统名称和版本
- **IP 地址**：设备当前或最后使用的 IP 地址
- **最后访问**：设备最后一次访问订阅的时间
- **访问次数**：设备访问订阅的总次数

### 3. 删除设备

**步骤：**
1. 在设备列表中找到要删除的设备
2. 点击设备卡片上的「移除」按钮（或「删除」按钮）
3. 系统会弹出确认对话框：
   - 提示：「确定要移除这个设备吗？移除后该设备将无法继续使用订阅服务。」
4. 点击「确定」按钮确认删除
5. 设备删除成功，设备数量减少

**删除结果：**
- 设备记录已从系统中删除
- 该设备无法再使用订阅服务
- 设备数量减少，可以添加新设备
- 如果设备数超过限制，删除设备后可以添加新设备

### 4. 删除限制

**权限限制：**
- 用户只能删除自己的设备
- 无法删除其他用户的设备
- 无法删除管理员手动添加的设备（如果存在）

**注意事项：**
- 删除设备后，该设备需要重新添加才能使用订阅
- 删除操作不可恢复
- 建议删除不再使用的设备，释放设备数量

## 二、管理员删除设备

### 1. 从用户详情删除

**路径：**
- **管理员后台** → **用户管理** → **用户列表** → 点击用户 → **用户详情** → **设备管理** 标签页

**步骤：**
1. 在用户列表中点击用户邮箱或用户名，打开用户详情对话框
2. 切换到「设备管理」标签页
3. 查看该用户的所有设备列表
4. 找到要删除的设备
5. 点击设备旁边的「删除」按钮
6. 确认删除操作

**删除结果：**
- 设备记录已删除
- 用户设备数量减少
- 该设备无法再使用订阅服务

### 2. 从订阅详情删除

**路径：**
- **管理员后台** → **订阅管理** → **订阅列表** → 点击订阅 → **订阅详情** → **设备列表**

**步骤：**
1. 在订阅列表中点击订阅，查看订阅详情
2. 找到「设备列表」或「设备管理」部分
3. 查看该订阅下的所有设备
4. 找到要删除的设备
5. 点击「删除」按钮
6. 确认删除操作

### 3. 批量删除设备

**说明：**
- 管理员可以在用户详情或订阅详情中批量删除设备
- 选中多个设备后，点击「批量删除」按钮
- 确认删除操作

**注意事项：**
- 批量删除会影响多个设备
- 删除操作不可恢复
- 建议谨慎操作

## 三、设备删除的影响

### 1. 对用户的影响

- **设备数量减少**：删除设备后，设备数量减少，可以添加新设备
- **设备无法使用**：删除的设备无法再使用订阅服务
- **需要重新添加**：如果设备仍在使用，需要重新添加才能继续使用

### 2. 对订阅的影响

- **设备数更新**：订阅的「当前设备数」会自动更新
- **设备限制**：如果设备数超过限制，删除设备后可以添加新设备
- **订阅状态**：删除设备不会影响订阅状态（正常/已过期/已禁用）

### 3. 对系统的影响

- **设备记录删除**：设备记录从数据库中删除
- **统计更新**：设备统计信息会自动更新
- **日志记录**：删除操作会记录到审计日志中

## 四、使用场景

### 场景 1：用户更换设备

**步骤：**
1. 用户更换了新设备
2. 旧设备不再使用
3. 用户登录系统，进入「设备管理」
4. 找到旧设备，点击「移除」按钮
5. 确认删除
6. 在新设备上添加订阅，自动识别为新设备

### 场景 2：设备数量超限

**步骤：**
1. 用户设备数达到限制（如 3/3）
2. 无法添加新设备
3. 用户查看设备列表，找到不常用的设备
4. 删除不常用的设备
5. 设备数量减少（如 2/3）
6. 可以添加新设备

### 场景 3：设备异常

**步骤：**
1. 用户发现设备列表中有异常设备（不认识的设备）
2. 可能是账户泄露或设备被误识别
3. 用户删除异常设备
4. 重置订阅地址（如果需要）
5. 重新添加正常设备

### 场景 4：管理员清理设备

**步骤：**
1. 管理员发现用户设备数异常（如 10/3）
2. 查看用户设备列表
3. 找到不活跃或异常的设备
4. 批量删除这些设备
5. 设备数量恢复正常

## 五、注意事项

1. **删除不可恢复**：删除设备后，设备记录无法恢复
2. **设备需要重新添加**：删除的设备需要重新添加才能使用订阅
3. **设备数量限制**：删除设备后，设备数量减少，可以添加新设备
4. **设备识别**：删除设备后，如果设备仍在使用，系统会重新识别为新设备
5. **批量删除**：管理员批量删除设备时，建议先确认设备列表，避免误删

## 六、常见问题

### Q1: 删除设备后，设备仍然可以使用？

**可能原因：**
- 设备删除后，如果设备仍在使用，系统会重新识别为新设备
- 如果设备数未超限，新设备可以正常添加
- 如果设备数已超限，新设备无法添加

**解决方法：**
- 删除设备后，建议重置订阅地址
- 或等待设备数量限制生效

### Q2: 如何批量删除设备？

**方法：**
- **用户端**：目前不支持批量删除，需要逐个删除
- **管理员端**：可以在用户详情或订阅详情中批量删除设备

### Q3: 删除设备后，设备数量没有减少？

**检查项：**
- 设备是否真的被删除（刷新页面查看）
- 是否有其他设备被添加
- 系统是否正常更新设备数量

**解决方法：**
- 刷新页面查看最新设备数量
- 检查设备列表，确认设备已删除
- 查看系统日志中的错误信息

### Q4: 用户无法删除设备？

**检查项：**
- 用户是否有权限删除设备
- 设备是否属于该用户
- 系统是否正常工作

**解决方法：**
- 确认用户登录状态
- 检查设备归属
- 联系管理员协助删除

### Q5: 管理员如何查看用户的所有设备？

**方法：**
1. 在用户列表中点击用户，查看用户详情
2. 切换到「设备管理」标签页
3. 可以看到该用户所有订阅下的所有设备
4. 可以按订阅筛选设备

## 七、设备删除与订阅重置的区别

### 设备删除

- **操作**：删除单个或多个设备
- **影响**：只影响被删除的设备
- **设备数量**：减少被删除的设备数量
- **其他设备**：不受影响，可以继续使用

### 订阅重置

- **操作**：重置订阅地址
- **影响**：清除所有设备记录
- **设备数量**：重置为 0
- **其他设备**：所有设备都无法使用，需要重新添加

**选择建议：**
- **删除设备**：只删除特定设备，其他设备不受影响
- **重置订阅**：清除所有设备，适合账户泄露或大量异常设备的情况
