# 登录历史管理功能说明

## 功能概述

登录历史功能用于记录和查看用户的登录记录，包括登录时间、IP 地址、地区信息、设备信息等。该功能帮助用户和管理员了解账户的登录情况，提高账户安全性。

## 访问路径

- **用户端** → **个人资料** → **登录历史**
- **管理员端** → **个人资料** → **安全设置** → **登录历史**

## 主要功能

### 1. 登录记录展示

登录时间、IP 地址/地区、设备信息（设备类型、操作系统、软件信息）、登录状态（成功/失败）。

### 2. 登录统计

总登录次数、不同 IP 数量、不同国家数量、距上次登录天数。

### 3. 地区信息显示

本地 IP、内网 IP、公网 IP 对应的显示方式；系统使用 GeoIP 数据库解析地理位置。
