# 数据分析功能说明

## 功能概述

数据分析功能为管理员提供系统数据的统计和分析，包括用户统计、收入统计、地区分析等。该功能帮助管理员了解系统运营情况，做出数据驱动的决策。

## 访问路径

- **管理员后台** → **数据分析**

## 主要功能

### 1. 数据概览

总用户数、活跃用户数、总订阅数、活跃订阅数、总收入、今日收入等。

### 2. 用户统计

用户增长趋势、用户状态分布、用户等级分布。

### 3. 收入统计

收入趋势、支付方式分布、套餐销售统计。

### 4. 地区分析

地区分布图、地区统计列表、地区详细统计。数据来源：登录历史、用户活动记录、审计日志；支持 GeoIP 解析。

### 5. 订阅与订单统计

订阅状态分布、订阅增长趋势、订单状态分布、订单趋势等。

## 注意事项

- 需要下载 GeoIP 数据库才能准确显示地区信息
- 地区信息基于 IP 解析，可能存在一定误差
