# 节点管理功能说明

## 功能概述

节点管理是管理员后台的核心功能，用于管理系统中所有的代理节点。节点可以通过自动采集或手动添加的方式获得，管理员可以对节点进行查看、编辑、删除、测试等操作。

## 访问路径

- **管理员后台** → **节点管理** → **节点管理**

## 主要功能

### 1. 节点信息展示

列表显示：节点ID、节点名称、类型（VMess、VLESS、Trojan、SS、SSR、Hysteria 等）、服务器、端口、地区、状态、来源（采集/手动）、操作。

### 2. 搜索和筛选

关键词搜索、类型筛选、地区筛选、状态筛选、来源筛选。

### 3. 节点添加

- **手动添加**：单个添加（输入节点链接）、批量导入（每行一个链接）
- **自动采集**：在「配置更新」页面配置节点源 URL，保存后点击「立即采集」或设置定时采集

### 4. 节点采集原理简述

1. 从配置的 URL 下载内容并解析节点链接（支持 vmess、vless、trojan、ss、ssr、hysteria、tuic、naive、anytls 等）
2. 解析节点信息并识别地区（从节点名称或服务器地址）
3. 按 Type:Server:Port 去重，已存在则更新，不存在则新建
4. 定时采集会清理所有「采集」来源的节点，手动添加的节点不会被打扫

### 5. 单个/批量操作

编辑节点、删除节点、测试节点、批量删除、批量测试。
