## 设备数量限制原理（核心功能）

设备数量限制是订阅管理的核心功能，详细原理请参考：

- [订阅管理功能说明 - 设备数量限制原理](./订阅管理功能说明.md#设备数量限制原理)
- [设备管理功能说明 - 设备数量限制原理](./设备管理功能说明.md#设备数量限制原理)

### 关键概念

1. **设备识别**：基于 User-Agent 和 IP 地址生成设备哈希
2. **设备漫游**：同一设备在不同网络环境下自动识别
3. **限制机制**：设备未超限/达到限制/超过限制的不同处理策略
4. **优先级策略**：采用"最近使用优先"策略，长时间未使用的设备自动淘汰
