# 邀请码管理功能说明

## 概述

邀请码管理功能用于创建和管理用户邀请码，支持邀请奖励系统。管理员可以查看所有邀请码、邀请关系、统计数据，并配置邀请奖励设置。

## 访问路径

- **管理员后台** → **邀请管理** → **邀请码列表** / **邀请关系**

## 一、邀请码列表管理

### 1. 查看邀请码列表

**功能：**
- 查看所有用户的邀请码
- 显示邀请码、邀请人、使用次数、奖励信息等
- 支持搜索、筛选、批量操作

**列表信息：**
- **邀请码**：邀请码字符串
- **邀请人**：创建邀请码的用户（用户名和邮箱）
- **已使用**：邀请码使用次数 / 最大使用次数
- **邀请人奖励**：邀请人获得的奖励金额（元）
- **被邀请人奖励**：被邀请人获得的奖励金额（元）
- **过期时间**：邀请码过期时间（永不过期显示为「永不过期」）
- **状态**：启用/禁用
- **创建时间**：邀请码创建时间

### 2. 搜索和筛选

**搜索方式：**
- **邀请人**：输入邀请人账号或邮箱搜索
- **邀请码**：输入邀请码搜索
- **状态筛选**：按启用/禁用状态筛选

**筛选选项：**
- **全部状态**：显示所有邀请码
- **启用**：只显示启用的邀请码
- **禁用**：只显示禁用的邀请码

### 3. 批量删除邀请码

**步骤：**
1. 在邀请码列表中勾选要删除的邀请码（可以多选）
2. 点击「批量删除」按钮
3. 确认删除操作

**注意事项：**
- 删除邀请码不会影响已建立的邀请关系
- 删除操作不可恢复
- 建议先禁用邀请码，确认无影响后再删除

## 二、邀请关系管理

### 1. 查看邀请关系

**功能：**
- 查看所有邀请关系（邀请人和被邀请人的关系）
- 显示邀请人、被邀请人、使用时间、奖励信息等
- 支持搜索、筛选、批量操作

**关系信息：**
- **邀请人**：创建邀请码的用户（用户名和邮箱）
- **被邀请人**：使用邀请码注册的用户（用户名和邮箱）
- **使用时间**：被邀请人使用邀请码的时间
- **邀请人奖励**：邀请人获得的奖励金额（元）
- **被邀请人奖励**：被邀请人获得的奖励金额（元）
- **订单金额**：被邀请人的首次订单金额（元）
- **奖励状态**：奖励是否已发放

### 2. 搜索和筛选

**搜索方式：**
- **邀请人**：输入邀请人账号或邮箱搜索
- **被邀请人**：输入被邀请人账号或邮箱搜索

**筛选功能：**
- 支持组合搜索（邀请人 + 被邀请人）
- 支持按奖励状态筛选

### 3. 批量删除邀请关系

**步骤：**
1. 在邀请关系列表中勾选要删除的关系（可以多选）
2. 点击「批量删除」按钮
3. 确认删除操作

**注意事项：**
- 删除邀请关系不会影响用户账户
- 删除操作不可恢复
- 建议谨慎操作

## 三、邀请奖励设置

### 1. 配置邀请奖励

**步骤：**
1. 在邀请管理页面，点击「邀请设置」按钮
2. 配置奖励金额：
   - **邀请人奖励**：当被邀请人首次购买套餐后，邀请人将获得的奖励金额（元）
   - **被邀请人奖励**：新用户使用邀请码注册后，立即获得的奖励金额（元）
3. 点击「保存设置」按钮

**设置说明：**
- 此设置将应用于所有新生成的邀请码
- 已生成的邀请码不受影响
- 奖励金额支持小数点后两位

### 2. 奖励发放机制

**邀请人奖励：**
- 触发条件：被邀请人首次购买套餐
- 发放时间：订单支付成功后立即发放
- 发放方式：自动添加到邀请人账户余额

**被邀请人奖励：**
- 触发条件：新用户使用邀请码注册
- 发放时间：注册成功后立即发放
- 发放方式：自动添加到被邀请人账户余额

### 3. 奖励设置示例

**示例配置：**
- **邀请人奖励**：10.00 元
- **被邀请人奖励**：5.00 元

**效果：**
- 新用户使用邀请码注册后，立即获得 5.00 元余额
- 新用户首次购买套餐后，邀请人获得 10.00 元余额

## 四、邀请统计

### 1. 统计数据

**统计信息：**
- **邀请码总数**：系统中所有邀请码的数量
- **邀请关系总数**：已建立的邀请关系数量
- **总奖励金额**：已发放的邀请奖励总额
- **被邀请人总消费**：被邀请人的累计消费金额

### 2. 查看统计

**位置：**
- 在邀请管理页面，切换到「统计」标签页
- 显示实时统计数据

## 五、邀请码功能说明

### 1. 邀请码生成

**生成方式：**
- 用户在用户端生成邀请码
- 管理员可以查看所有邀请码
- 每个用户可以有多个邀请码

**邀请码属性：**
- **使用次数限制**：可以设置最大使用次数（可选）
- **过期时间**：可以设置过期时间（可选）
- **奖励设置**：可以设置邀请人和被邀请人奖励（可选）
- **状态**：启用/禁用

### 2. 邀请码使用

**使用流程：**
1. 新用户注册时输入邀请码
2. 系统验证邀请码有效性
3. 建立邀请关系
4. 发放被邀请人奖励（如果配置）
5. 被邀请人首次购买套餐后，发放邀请人奖励（如果配置）

**验证规则：**
- 邀请码必须存在且启用
- 邀请码未过期（如果设置了过期时间）
- 邀请码未达到使用次数限制（如果设置了限制）
- 邀请码未被禁用

### 3. 邀请关系建立

**建立条件：**
- 新用户使用有效的邀请码注册
- 系统自动建立邀请关系
- 记录邀请人和被邀请人信息

**关系信息：**
- 邀请人 ID
- 被邀请人 ID
- 使用时间
- 奖励信息

## 六、使用场景

### 场景 1：配置邀请奖励

```
1. 点击「邀请设置」按钮
2. 设置邀请人奖励：10.00 元
3. 设置被邀请人奖励：5.00 元
4. 点击「保存设置」
5. 新生成的邀请码将使用此奖励设置
```

### 场景 2：查看邀请效果

```
1. 切换到「统计」标签页
2. 查看邀请码总数、邀请关系总数
3. 查看总奖励金额、被邀请人总消费
4. 评估邀请活动的效果
```

### 场景 3：管理异常邀请码

```
1. 搜索异常邀请码（如：使用次数异常）
2. 禁用或删除异常邀请码
3. 查看相关邀请关系
4. 处理异常情况
```

## 七、注意事项

1. **奖励设置**：修改奖励设置只影响新生成的邀请码，已生成的邀请码不受影响
2. **删除邀请码**：删除邀请码不会影响已建立的邀请关系
3. **删除邀请关系**：删除邀请关系不会影响用户账户，但会丢失奖励记录
4. **奖励发放**：奖励会自动发放到用户账户余额，无需手动操作
5. **邀请码限制**：可以设置使用次数限制和过期时间，用于控制邀请码的有效性

## 八、常见问题

### Q1: 如何修改已生成邀请码的奖励？

**答：**
- 已生成的邀请码奖励无法修改
- 只能修改新生成的邀请码的奖励设置
- 建议在生成邀请码前配置好奖励设置

### Q2: 邀请码可以设置使用次数限制吗？

**答：**
- 可以。用户在生成邀请码时可以设置最大使用次数
- 达到使用次数后，邀请码将无法使用
- 管理员可以查看邀请码的使用次数

### Q3: 邀请码可以设置过期时间吗？

**答：**
- 可以。用户在生成邀请码时可以设置过期时间
- 过期后，邀请码将无法使用
- 管理员可以查看邀请码的过期时间

### Q4: 如何查看邀请效果？

**方法：**
- 切换到「统计」标签页
- 查看邀请码总数、邀请关系总数
- 查看总奖励金额、被邀请人总消费
- 评估邀请活动的效果

### Q5: 删除邀请关系会影响用户账户吗？

**答：**
- 不会。删除邀请关系不会影响用户账户
- 但会丢失奖励记录
- 建议谨慎操作
