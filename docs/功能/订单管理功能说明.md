# 订单管理功能说明

## 功能概述

订单管理功能用于管理系统中的所有订单记录，包括用户订单和管理员订单查看。订单记录了用户的套餐购买、充值等交易信息。

## 访问路径

- **管理员后台** → **订单管理** → **订单列表**
- **用户端** → **订单记录**

## 主要功能

### 1. 订单信息展示

订单号、用户信息、套餐信息、支付方式、订单金额、订单状态（待支付/已支付/已取消）、创建时间、支付时间、操作。

### 2. 搜索和筛选

关键词搜索（订单号、邮箱、用户名）、状态筛选、支付方式筛选、时间范围筛选。

### 3. 订单操作

查看详情、标记为已支付（线下支付等）、取消订单、导出订单（CSV）。

### 4. 订单状态

- **pending**：待支付，可取消或手动标记为已支付
- **paid**：已支付，用户余额或订阅已更新
- **cancelled**：已取消
