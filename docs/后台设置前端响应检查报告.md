## 🔍 已修复的问题

### 问题1: 字段名不匹配
- **问题**: 后端返回下划线格式（`registration_enabled`），前端使用驼峰格式（`allowRegistration`）
- **修复**: 所有相关文件已支持多种字段名格式
- **影响范围**: 
  - `registration_enabled` / `allowRegistration`
  - `min_password_length` / `minPasswordLength`
  - `invite_code_required` / `inviteCodeRequired`
  - `email_verification_required` / `emailVerificationRequired`

### 问题2: 邀请码必填标记缺失
- **问题**: 邀请码字段缺少 `:required` 属性，不显示红色星号
- **修复**: 已添加 `:required="inviteCodeRequired"` 属性
- **影响文件**: 
  - `frontend/src/views/UnifiedAuth.vue`
  - `frontend/src/views/Register.vue`
